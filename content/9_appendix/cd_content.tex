\chapter{CD content}

\begin{description}
\item[Documentation] can be found in the folder \custtt{documentation}
\item[Internal documents] can be found in the folder \custtt{internal\_docs}
\item[Software] can be found in the folder \custtt{source} and contains the following directories.
\end{description}

\begin{itemize}
\item build - Make-/build files. IDE project files.
\item include - Header files that expose the public interface to applications using it.
\item lib - External libraries source.
\item src - Project source (including .h headers that are not supposed to be exposed publicly).
\item tests - Test programs.
\end{itemize}
